# クラウド版

<Steps>
  <Step title="ワークスペースを作成する">
    ワークスペースを作成して、Editorを開きます。
  </Step>
  <Step title="データ処理をPythonで実装する">
    データの整形・データ分析の実行・チャートの作成などをPythonで実装します。必要に応じてSQLを使うこともできます。

    ```python src/python/data.py
    import pandas as pd

    @morph.func(name='data')
    def main():
      df = pd.read_csv('data.csv')
      return df

    ```
  </Step>
  <Step title="MDXファイルを編集する">
    MDXファイルに、説明文やコンポーネントを追加して、アプリを完成させます。

    ```md src/pages/index.mdx
    # Data Application

    <DataTable loadData='data' />
    ```
  </Step>
  <Step title="Pageタブ または Viewerモードでアプリを使う">
    Editor画面の "Page" タブ または Viewerモードでアプリを使うことができます。
  </Step>
</Steps>
