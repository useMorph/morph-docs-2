# MetricsGrid

MetricsGridコンポーネントは、Python関数やSQLファイルの実行結果の各行を、Metricsコンポーネントで表示します。

表示はグリッド形式です。

```tsx
<MetricsGrid 
  alias="aggregate_py" 
  variables={{
    department
  }}
  cols="3"
  valueKey="amount" 
  labelKey="name" 
/>
```

## プロパティ

<ParamField path="alias" type="string" required>
  Python関数やSQLファイルの name
</ParamField>

<ParamField path="varialbes" type="{ [key: string]: Variable }">
  Python関数やSQLファイルに渡す変数。 `variable()`関数 を使って宣言してください。
</ParamField>

<ParamField path="valueKey" type="string">
  値を表示するカラム名
</ParamField>

<ParamField path="labelKey" type="string">
  値の上部に表示される文字列をデータの中から取得したい場合、そのカラム名
</ParamField>

<ParamField path="label" type="string">
  値の上部に表示される文字列
</ParamField>

<ParamField path="unit" type="string">
  値の右部に表示される文字列
</ParamField>

# Metrics

Metricsコンポーネントは、Python関数やSQLファイルの実行結果のうち、**1行目**の特定の列の値を表示します。

```tsx
<Metrics 
  alias="aggregate_py"
  variables={{
    department
  }}
  valueKey="amount" 
  unit="USD"
  label="Total Sales"
/>
```
