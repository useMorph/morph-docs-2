DataTableコンポーネントは、Python関数やSQLファイルの実行結果をテーブル表示します。

```tsx
<DataTable 
  alias="aggregate_py" 
  variable={{
    department
  }}
  hederKeys={['name', 'email', 'department']}
  height={300} 
/>
```

## プロパティ

<ParamField path="alias" type="string" required>
  Python関数やSQLファイルの name
</ParamField>

<ParamField path="varialbes" type="{ [key: string]: Variable }">
  Python関数やSQLファイルに渡す変数。 `variable()`関数 を使って宣言してください。
</ParamField>

<ParamField path="headerKeys" type="string[]">
  表示するカラムの名前。順番も指定できます。
</ParamField>

<ParamField path="width" type="number">
  テーブルの横幅。指定がない場合は親要素に合わせます。
</ParamField>
