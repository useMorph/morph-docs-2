---
title: 'Updating the Library'
---

Currently, the library for building pages is being actively developed, and new features are being added. By updating the library, you can make use of these new functionalities.

This section explains how to update the library.

## Running the Command

You can update the library by running the following command in the terminal. This may take about **5 minutes**.

```bash Shell
npx morph-page clean && npm i -S @use-morph/page-build && npx morph-page init
```
