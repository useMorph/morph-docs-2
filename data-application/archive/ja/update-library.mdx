---
title: 'ライブラリのアップデート'
---

現在、ページ構築のためのライブラリは積極的に開発されており、新しい機能が追加されています。ライブラリのアップデートを行うことで、新しい機能を利用することができます。

本セクションでは、ライブラリのアップデート方法について説明します。

## コマンドの実行

以下のコマンドをターミナルで実行することで、ライブラリのアップデートを行うことができます。これには、 **5分** 程度の時間がかかる場合があります。

```bash Shell
npx morph-page clean && npm i -S @use-morph/page-build && npx morph-page init
```
