# 問題

このセクションでは、以下の画像のようにEmbedやDataTableなどのすべてのコンポーネントが`Unexpected end of JSON input`エラーを表示する場合の解決策について説明します。

<img
    src="/assets/images/troubleshooting/mdx-api-error.png"
    alt="MDX API Error"
/>

# 解決策

Morphでは、MDXからSQL/Pythonを実行するためにワークスペース内でAPIサーバーが起動されます。このエラーは、APIサーバーがクラッシュしたか、何らかの理由で起動していない場合に発生します。

通常、APIサーバーはクラッシュした場合に自動的に復旧するように設定されていますが、サーバーが起動に失敗した場合にこのエラーが発生することがあります。

この問題を解決するには、ターミナルで以下のコマンドを実行してAPIサーバーを再起動し、エラーを修正してください。

```bash Shell
morph serve --restart
```
