## October 24, 2024

### New Component and updates for Pages v0.4.19

- Added new layout component `Tabs` `Accordion` `Alert`. [Documentation](/data-application/en/layout-components)

- Added `headerKeys` property to specify which columns to display in a `DataTable`. [Documentation](/data-application/en/data-components)

- Added `withClearButton` property to `VariableSelect`. [Documentation](/data-application/en/use-variables)

---

## October 22, 2024

### New Component for Pages v0.4.15

- Added `MetricsGrid` component to display metrics in grid format.

- Added `VaraibleSelectItems` component as a child of `VariableSelect`. You can create dynamic select boxes by passing data retrieved with the `getJson` function.

- The `column` property of the `Metrics` component has been changed to `valueKey`.

- Added `getJson` and `value` functions to retrieve and display data in MDX files.

[Documentation](/data-application/en/data-components)

## October 15, 2024

### MDX Library Update

- Added [Metrics component](/data-application/en/data-components#metrics) as a data component.
- Added layout components such as [Grid](/data-application/en/layout-components#grid) and [Panel](/data-application/en/layout-components#panel).
- Variables can now be directly defined in mdx files using functions like `variable`. See the [documentation](/data-application/en/use-variables) for usage instructions.
- Added input components such as [VariableInput](/data-application/en/use-variables#variableinput), [VariableDatePicker](/data-application/en/use-variables#variabledatepicker), and [VariableSelect](/data-application/en/use-variables#variableselect).

## October 8, 2024

### Default Values for `@morph.variables`

- Default values can now be set using [`@morph.variables("var_name", default="default_value")`](/sql-python/en/python/morph-variables), allowing a default value to be used if no argument is passed during execution.
