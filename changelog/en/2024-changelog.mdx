<Update label="2024-12-20">
### New Integration

- Added **MS SQL Server** as a connection destination for SQL Connection.

### Support for Scheduled Jobs

- The scheduled Jobs feature, previously available only in the cloud version, can now be configured in the framework's morph_project.yml. [Documentation](/docs/en/develop/guides/job-schedule)

</Update>

<Update label="2024-12-18">

### OSS Version Released

Released the OSS version of the Morph framework for use in local environments.

[Github repository](https://github.com/morph-data/morph)

- Integrated the SQL, Python, and Pages frameworks into a single framework.
- Enabled the `morph serve` command to launch data apps in development mode in local environments.
- Enabled the `morph sync` command to deploy data apps built in local environments to the cloud.

</Update>

<Update label="2024-12-17">
### New AI Functions

Added shorthand functions to easily build AI pipelines.

- Added `text_to_sql` function. [Documentation](/reference/en/morph_lib/ai/text_to_sql)
- Added `text_to_matplotlib` function. [Documentation](/reference/en/morph_lib/ai/text_to_matplotlib)
- Added `text_to_plotly` function. [Documentation](/reference/en/morph_lib/ai/text_to_plotly)
- Added `text_to_html` function. [Documentation](/reference/en/morph_lib/ai/text_to_html)
- Added `text_to_markdown` function. [Documentation](/reference/en/morph_lib/ai/text_to_markdown)

</Update>

<Update label="2024-12-16">
### New Components for Pages

- Added `LLM` component for building LLM apps such as chat and text generation. [Documentation](/data-application/en/ai-component-llm)

### Added Streaming Function to Python SDK

- Added `morph_lib.stream.create_chunk` function to return function results in text streaming format. [Documentation](/reference/en/morph_lib/stream/create_chunk)

</Update>

<Update label="2024-10-24">

### New Component and updates for Pages v0.4.19

- Added new layout component `Tabs` `Accordion` `Alert`. [Documentation](/data-application/en/layout-components)

- Added `headerKeys` property to specify which columns to display in a `DataTable`. [Documentation](/data-application/en/data-components)

- Added `withClearButton` property to `VariableSelect`. [Documentation](/data-application/en/use-variables)

</Update>

<Update label="2024-10-22">

### New Component for Pages v0.4.15

- Added `MetricsGrid` component to display metrics in grid format.

- Added `VaraibleSelectItems` component as a child of `VariableSelect`. You can create dynamic select boxes by passing data retrieved with the `getJson` function.

- The `column` property of the `Metrics` component has been changed to `valueKey`.

- Added `getJson` and `value` functions to retrieve and display data in MDX files.

[Documentation](/data-application/en/data-components)

</Update>

<Update label="2024-10-15">

### MDX Library Update

- Added [Metrics component](/data-application/en/data-components#metrics) as a data component.
- Added layout components such as [Grid](/data-application/en/layout-components#grid) and [Panel](/data-application/en/layout-components#panel).
- Variables can now be directly defined in mdx files using functions like `variable`. See the [documentation](/data-application/en/use-variables) for usage instructions.
- Added input components such as [VariableInput](/data-application/en/use-variables#variableinput), [VariableDatePicker](/data-application/en/use-variables#variabledatepicker), and [VariableSelect](/data-application/en/use-variables#variableselect).

</Update>

<Update label="2024-10-08">

### Default Values for `@morph.variables`

- Default values can now be set using [`@morph.variables("var_name", default="default_value")`](/sql-python/en/python/morph-variables), allowing a default value to be used if no argument is passed during execution.

</Update>
