AWS Athenaへの接続を設定するには、ホーム画面の「Connections」タブを開いて、「Create New Connection」ボタンから接続設定モーダルを開き、Athenaを選択します。

## **接続設定項目 (Account)**

> ここではAccountセクションの入力内容について説明します。必要に応じて以下の接続設定項目を入力して、右下に表示されるSubmitボタンを押下してください。

1. **Access key**
    - AWSアカウントのアクセスキー
2. **Secret key**
    - AWSアカウントのシークレットキー
3. **Region**
    - AWSリージョン

![Athena Account](/assets/images/integrations/athena_account.png)

### 接続設定を完了する

> Accountセクションの入力が終わって画面右下のSubmitボタンを押下したら、次はConnection slugを入力して、接続設定を完了してください。

1. **Catalog**
    - AWS Athenaのデータカタログ名
2. **Database**
    - AWS Athenaのデータベース名
3. **Work group**
    - AWS Athenaのワークグループ名

![Athena Setting](/assets/images/integrations/athena_setting.png)

Connections一覧に接続設定が表示されたら設定完了となります。

![Athena Complete](/assets/images/integrations/athenacomplete.png)
