title: Stripe

Stripe is a powerful tool that simplifies online payment processing, offering a seamless payment solution that can be easily integrated into websites and applications.

By integrating Morph with Stripe, users can visualize Stripe data and build complex finance-focused data applications by combining it with other data sources.

## How to use

It can be created from the workspace by going Connections > Integrations.
