title: Linear
---

Linear is a tool designed to streamline project and task management, allowing users to organize team workflows and track progress. With an intuitive interface and powerful features, Linear enhances team productivity.

By integrating Morph with Linear, users can easily create dashboards and reports that leverage data from Linear.

## How to use

It can be created from the workspace by going Connections > Integrations.
