title: Airtable
---

Airtable is a cloud-based collaboration tool that combines the ease of use of spreadsheets with the powerful capabilities of a database. Users can create customizable tables, share, and edit data in real time. Additionally, Airtable can integrate with numerous third-party applications, enabling workflow automation.

By integrating Morph with Airtable, you can build data applications through data visualization on Airtable and integration with other data sources.

## How to use

It can be created from the workspace by going Connections > Integrations.
