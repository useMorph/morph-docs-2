Retrieves the access token obtained from the Integration.

```python
from morph_lib.api import get_auth_token

get_auth_token(
	connection: str
) -> str
```

### Parameters

<ParamField body="connection" type="str" required>
Specify the name of the Integration registered in the screen.
</ParamField>
