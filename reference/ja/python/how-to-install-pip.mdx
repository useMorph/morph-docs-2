---
title: 'pipによるPythonパッケージのインストール方法'
---

Morphでは、Pythonで外部モジュールをしたい場合にpipを使用してパッケージをインストールすることができます。

## Morph上のpipの利用方法

1. インストールしたいパッケージの名前を特定します。

    例えばopenaiのAPIクライアントを利用したい場合、 `openai` というパッケージ名で引き当てることができます。
    これはchatgptなどLLMに聞けば特定できる他、[こちら](https://pypi.org/)で検索ができます。

2. エディタ上で、Control + ` を押してターミナルを開きます。
3. ターミナル上で下記コマンドを実行します

    `pip install {パッケージ名}`

    `{パッケージ名}` は1で決めたものを指定してください。

4. コードから、該当するパッケージが利用できることを確認します。

    一度インストールすると、そのワークスペースでは次回起動時も当該パッケージを引き続き利用することができます. 別のワークスペースで利用するためには、別途インストールをする必要があります。
