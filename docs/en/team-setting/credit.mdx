---
title: 'Credit'
---

Morph manages the execution of cloud machines to run workspaces and AI functions by consuming credits.

A certain number of credits are packaged with each usage plan and additional credits can be purchased.

<Warning>The credits purchased are valid for 1 month.</Warning>

## Credit usage

Each function consumes credits according to the table below:

### VM execution

Morph's workspace runs on a Virtual Machine (VM). When the VM is running, credits are consumed.

| CPU | Memory | Credit consumption / 1 hour |
| :--- | :--- | :--- |
| 2vCPU | 4MB | 30 |

### AI function

MorphAI's functions consume credits according to the table below:

| AI function | Credit consumption |
| :--- | :--- |
| [Code Generation](/docs/en/morph-ai/code-generation) | 6 |
| [Coding Assistant](/docs/en/morph-ai/code-assistant) | 6 |
| [Data Pipeline Builder](/docs/en/morph-ai/data-pipeline-builder) | 18 |

## Credit management

You can check how much credit you have used, and purchase additional credits from the ‘Credit’ tab on the home screen.
