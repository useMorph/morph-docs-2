Morphでは、ワークスペースを実行するためのクラウドマシンの実行やAI機能の実行をクレジットの消費によって管理しています。

利用プランごとに一定数のクレジットがパッケージされ、追加購入することも可能です。

<Warning>追加購入をしたクレジットの有効期限は1ヶ月です。</Warning>

## クレジットの消費

各機能は、以下の表に従ってクレジットを消費します。

### VMの実行

Morphのワークスペースは、Virtual Machine(VM)が稼働しています。VMの稼働時にはクレジットが消費されます。

| インスタンスタイプ | CPU | メモリ | 消費クレジット / 1時間 |
| :--- | :--- | :--- | :--- |
| Standard | 2vCPU | 4GB | 100 |
| Large | 4vCPU | 8GB | 200 |
| XLarge | 8vCPU | 16GB | 400 |

### データ容量

ビルトインPostgreSQLとVM上のディスク容量の大きさに応じて、月末にクレジット消費がされます。

| サービス | 消費クレジット / GiB |
| :--- | :--- |
| ビルトインPostgreSQL | 300 |
| VM容量 | 300 |

<Warning>
ビルトインPostgreSQLの容量は、無料プランでは0.5GiBまでの利用に制限されています。

Proプランでは、5GiBまではクレジット消費はされません。
</Warning>

### AI機能

MorphAIの機能では、以下の表に従ってクレジットを消費します。

| AI機能 | 消費クレジット |
| :--- | :--- |
| [コード生成](/docs/ja/morph-ai/code-generation) | 6 |
| [コーディングアシスタント](/docs/ja/morph-ai/code-assistant) | 6 |
| [データパイプラインの自動構築](/docs/ja/morph-ai/data-pipeline-builder) | 18 |

## クレジットの管理

ホーム画面の「Credit」タブからクレジットの使用量の確認と追加購入ができます。
