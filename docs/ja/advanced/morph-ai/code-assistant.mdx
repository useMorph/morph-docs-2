コーディングアシスタントは、コーディングをサポートする機能です。

エディター上からファイルを開くと、画面右のタブからMorph AIにアクセスをすることができます。
現時点で公式でサポートをしている拡張子は以下になります。これらのファイルをエディターで開いた場合には、自動的にMorph AIはコーディングアシスタントとして動作します。

- Python (.py)
- SQL (.sql)
- MDX (.mdx)
- TypeScript JSX (.tsx)

コーディングアシスタントは、現在のファイルの中身やそれらが参照しているデータソースの構造を理解して、コードの修正やリファクタリングを行うことができます。
