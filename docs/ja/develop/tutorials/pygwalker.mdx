Morphのワークスペースは、以下の4つのセクションから構成されます。

- Code: VS Codeベースのエディターと独自のフレームワークにより、データアプリケーションの構築、データ分析を高速に実施できます。
- Data: Built-in PostgreSQLを使用することができます。また、RDB / データウェアハウス / SaaSとの連携を作成し管理することができます。
- Pages: 構築したデータアプリケーションにアクセスすることができます。
- Jobs: スケジュール実行の設定や実行ログの確認を行うことができます。
