Sample data apps are a collection of use cases using Morph.

When creating a workspace, you can use sample data apps as templates.
If you want to use specific datasets, using sample apps can greatly reduce the initial setup effort and time to utilization.

### About new sample data apps

If you have any requests for sample apps, please feel free to contact the Morph team.

Please contact us [here](/troubleshooting/en/support/contact-support).
