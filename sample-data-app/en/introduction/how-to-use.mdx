title: 'How to use Sample Data Apps'
---

Sample data apps can help you easily build more complex data apps and data pipelines.

You can check the list of sample data apps at the same time by logging in to Morph and accessing the workspace list page.

By selecting a sample data app from the list, you can create a new workspace from the selected sample.

`how-to-use.md` contains the steps to set up a sample data app.
After creating a workspace, please follow the steps to set up.

After setting up, you can immediately use the data app.
